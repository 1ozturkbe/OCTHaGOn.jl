Main.OCTHaGOn.GlobalModel — Type

Contains all constraints and data to solve the global optimization optimization problem. Add linear constraints to a GlobalModel through JuMP.@constraint(gm.model, ...), and nonlinear constraints using add_nonlinear_constraint.

source
Main.OCTHaGOn.BlackBoxLearner — Type

Union of BlackBoxClassifier and BlackBoxRegressor types.

source
Main.OCTHaGOn.BlackBoxClassifier — Type

Allows for approximation of constraints using OCTs. Can be added to GlobalModel using add_nonlinear_constraint or add_nonlinear_or_compatible.

source
Main.OCTHaGOn.BlackBoxRegressor — Type

Allows for approximation of continuous functions or objectives using ORTs. Can be added to GlobalModel using add_nonlinear_constraint or add_nonlinear_or_compatible.

source

source
Main.OCTHaGOn.LinkedClassifier — Type

Contains data for a constraint that is repeated. Add to GlobalModel using add_linked_constraint.
